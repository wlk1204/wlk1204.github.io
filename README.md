### 前端工程师的三条出路

第一条：和团队做出一个很好的产品，拿很多分红；

第二条：在某个技术深究，做一个你比别人都厉害的佼佼者；

第三条：具备很强的学习能力，每个领域都要懂，但可以不用太深入，能当好 leader 做好技术选型。
