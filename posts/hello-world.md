# Hello World

欢迎来到我的个人博客！

这是使用 [VitePress](https://vitepress.dev/) 搭建的博客站点，部署在 GitHub Pages 上。

## 功能特性

- 首页展示最新文章
- 标签分类浏览
- RSS 订阅
- 深色/浅色主题切换
- 站内全文搜索

## 开始写作

在 `posts/` 目录下创建 Markdown 文件即可发布新文章。Frontmatter 示例：

```yaml
---
title: 文章标题
date: 2026-07-11
tags: [标签一, 标签二]
description: 文章摘要
publish: true
---
```

将 `publish` 设为 `false` 可以隐藏草稿，不会出现在列表和 RSS 中。

祝你写作愉快！
